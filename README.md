## Description

This project is a front-end web application that replicates the core functionality of Google Search. It includes pages for Google Search, Google Image Search, and Google Advanced Search, allowing users to submit search queries that redirect to Google with the appropriate URL parameters. The project was developed as part of CS50W Project 0 to practice HTML form design, CSS styling, and basic JavaScript for handling user interactions and form behavior.
